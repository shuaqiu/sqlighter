# sqlighter
a SQLite helper lib

## usage

1. Add the JitPack repository to your build file

  Add it in your build.gradle at the end of repositories:

  ```
  repositories {
    // ...
    maven { url "https://jitpack.io" }
  }
  ```
  
1. Add the dependency in the form

  ```
  dependencies {
    // ...
    compile 'com.github.shuaqiu:sqlighter:1.0'
  }
	```
